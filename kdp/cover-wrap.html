📋 KDP Cover Requirements & Instructions

Before uploading to KDP:

  • Page Count: Use KDP's cover calculator with your exact page count. This determines the spine width.
  • Spine Width: Update the field above with the value from KDP's calculator, then reprint.
  • Bleed: Layout includes 0.125" bleed on outer edges (as per KDP specs).
  • Barcode: KDP will automatically add a barcode on the back cover. Leave the "Barcode zone" blank.
  • Colors: CMYK recommended for color printing. PDF preset: "Press Quality".
  • Margins: Do NOT add additional margins—they reduce usable space.

⚠️ After finalizing interior PDF page count:
1. Calculate spine width using KDP's free calculator
2. Update spine width field above
3. Print this page again to PDF
4. Upload both interior & cover PDFs to KDP
