## 📞 TROUBLESHOOTING

### Website shows blank
- Make sure all files in correct folder
- Open index.html from same folder as src/
- Check browser console (F12) for errors

### Entries not showing
- Check daily-data.js syntax (JavaScript errors)
- Verify date format: "MM-DD"
- Save and refresh browser

### Export not working
- Check browser JavaScript is enabled
- Try different browser
- Clear cache and reload

### Print looks bad
- Use Firefox or Chrome (not Edge)
- Print as PDF (Chrome → Save as PDF)
- Check print.css loaded correctly
